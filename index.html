SteerDuplex

Controllable Bidirectional Text Generation with Preference Steering

Abstract

SteerDuplex presents a novel approach to controllable text generation that leverages bidirectional modeling for enhanced control and coherence. This work introduces methods for steering language models in both forward and backward directions, enabling fine-grained control over generated content through preference-based constraints.

Key Innovation: A dual-directional generation framework that combines forward and backward language modeling to achieve superior controllability while maintaining natural language quality.

Main Contributions

📊 Bidirectional Architecture

Novel framework combining forward and backward language modeling for improved control mechanisms.

🎯 Preference Steering

Efficient methods for incorporating user preferences and constraints into the generation process.

✨ Quality & Controllability

Achieves high-quality text generation while maintaining precise control over output characteristics.

🔬 Comprehensive Evaluation

Extensive experimental validation on multiple tasks and datasets demonstrating effectiveness.

Technical Approach

Bidirectional Generation Framework

SteerDuplex leverages a dual-model architecture that processes text bidirectionally. The forward model generates content while the backward model provides contextual constraints, enabling sophisticated control mechanisms.

Preference Steering Mechanism

The steering mechanism allows users to specify high-level preferences and constraints that guide the generation process. This is achieved through a preference encoding module that integrates user specifications into the generation pipeline.

Key Technical Features

  • Efficient bidirectional inference mechanism
  • Preference encoding and integration layer
  • Joint optimization of forward and backward models
  • Scalable implementation for large language models

Experimental Results

Our experiments demonstrate significant improvements in controllability while maintaining or improving generation quality:

  • Controllability Metrics: Superior constraint satisfaction compared to baseline methods
  • Quality Metrics: Competitive or improved BLEU, ROUGE, and human evaluation scores
  • Efficiency: Reasonable computational overhead compared to standard generation
  • Scalability: Demonstrated effectiveness across various model sizes and domains
